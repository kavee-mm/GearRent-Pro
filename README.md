# GearRentPro

## Description
GearRentPro is a Java-based equipment rental management system developed using
Java 8, JDBC, and MySQL. The project follows a layered architecture using
Controller, Service, DAO, and Entity layers.

## Technologies Used
- Java 8 (Zulu)
- Apache NetBeans IDE
- MySQL 8.x
- JDBC (MySQL Connector/J)

## How to Run
1. Install Java 8
2. Install MySQL and create database `gearrentpro`
3. Add MySQL Connector/J to project libraries
4. Configure `DBConnection.java`
5. Run the project using NetBeans

## Note
Some features are under development.
